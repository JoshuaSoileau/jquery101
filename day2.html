Building a Robotic Overlord.

If you have not already done so, download the Day 2 Course Materials here.

Okay jQueriers, it's time to get for reals. We're gonna make a full up game out of this, building on everything we've done to this point.

  • add click events to all `box` elements to:
    • `hide` them
    • `remove` them from the DOM
    • after `remove`ing them, increment the number inside `boxes destroyed` box
  • write a function to insert new `box` elements into the `section` tag with the class `boxes`. Make it a full on, callable function
  • wrap that function in some sort of javascript thing that will run repeatedly every few seconds (google!).
  • if we get here in time, write some javascript logic that will check if there are any boxes remaining, and if not, change the text on the page to say that you've won the game!
