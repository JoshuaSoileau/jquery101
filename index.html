CODEcamp - jQuery 101

Course Overview

Why should I take jQuery?

jQuery is one of the most popular javascript libraries on the planet. jQuery is the easiest way to get complicated javascript interaction logic up and running. By taking this class, you'll be making your start getting into the frontend development world and learning valuable skills to make you great at frontend coding. In addition to learning the ins & of jQuery, we'll learn how to use Google Chrome's developer tools to debug your frontend code and make changes to your site in a quick & efficient manner.

What will I learn?

jQuery is a collection of commonly used javascript widgets and functions that has more or less taken the internet by storm. jQuery makes it easy to add interactivity to your webpages. Accordians, sliders, repositioning, animations, if you want to learn how to make interactive web pages that users will love, you'll learn it all right here in jQuery 101.

What can I expect?

Each 2.5-hour session is a mixture of Lecture and hands-on Exercises to best teach you the fundamentals of jQuery. Getting started with jQuery can be easy or challenging, depending on your experience with JavaScript, HTML, CSS, and programming concepts in general. While we will learn all of the fundementals of building websites with jQuery, to become great at coding with jQuery will take practice and hard work. Students are expected to continue working & learning outside of class in addition to the exercises we complete in our 2.5 hour sessions.

Class 1 - Intro to jQuery, Using Selectors, the DOM, and Events

In your first class you will cover the basics of jQuery. We'll start with what jQuery is and how it relates to javascript. We'll learn how to understand on a high level what the jQuery() function is and does.

We're going to dig into the DOM at a pretty deep level in this class. After we understand what jQuery is, we'll learn how to use it to manipulate DOM elements, and how using the jQuery object creator gives us the ability to quickly and easily make changes in the DOM in ways that can be long and confusing in base javascript.

After we are pros at grabbing and editting DOM elements, we're going to learn about Events. What they are, how they work, how they travel, and how we can watch them and their effect on DOM elements.
